# Express Crash Course

## Quick Start
  Install dependencies:

```bash
$ npm install
```

  Start the server:

```bash
$ npm start
```
  View the website at: http://localhost:3000

## Examples api

  GET:
```bash
$ http://localhost:3000/apis/restaurants
```  
  GET by id:
```bash
$ http://localhost:3000/apis/restaurants/:id
```
